# VROOM 2026 Landing Page

## How to host on GitHub Pages

1. Create a new GitHub repository.
2. Upload these files:
   - `index.html`
   - `style.css`
   - `script.js`
   - `assets/vroom-logo.png`
3. Go to Repository Settings.
4. Open **Pages**.
5. Under **Build and deployment**, select:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
6. Save.
7. GitHub will give you a live website URL.

## Must edit before publishing

In `index.html`, replace:

- `PAYMENT_LINK_HERE` with your Razorpay/payment link.
- `WHATSAPP_LINK_HERE` with your WhatsApp click-to-chat link.
